updatebib
=======

### Installation
```bash
git clone https://github.com/srossd/updatebib.git
cd updatebib
npm install
```

### Run Development
```bash
npm run dev
open http://localhost:3000
```

### Create Distribution
```bash
npm run build
```

### Run Distribution
```bash
npm start
open http://localhost:3001
```
